# TalaSitio
🏘️ TalaSitio
A Web-Based Census and Information Update System for Sitio Communities
📌 Overview
TalaSitio is a web-based census and resident information management system designed for small communities such as sitios and barangays.
The system allows:
Administrators to manage resident records
Residents to request updates or corrections to their personal information
Secure and organized storage of community data
This project aims to improve efficiency, accuracy, and accessibility of local census records.

🎯 Objectives
Digitize resident census records
Minimize manual paperwork
Allow residents to submit update/edit requests online
Provide administrators with an easy-to-use dashboard
Maintain accurate and up-to-date community data
